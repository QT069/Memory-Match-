# Memory-Match
1. Engine sử dụng: UNITY
2. Thành viên hiện tại:
   > **Tô Phú Quý**
   > **Trần Thiên Quang**
3. Tính năng dự kiến:
   > 2 màn chơi
     - 4x4 : 5 cặp
     - 6x6 : 10 cặp 
   > Giới hạn số cặp **được chọn sai** của mỗi màn chơi 
   > Tính điểm (1-2-3 sao) theo mốc thời gian  
   > Lưu kỷ lục 
   > *(Vật phẩm hỗ trợ nâng thời gian, nâng điểm...) cập nhật sau nếu được*
